# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in Aeris PromptShield, please report it responsibly:

**Email:** aeris-ai@proton.me

**Subject:** `[SECURITY] Brief description`

**Include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment:** Within 24 hours
- **Initial assessment:** Within 72 hours
- **Fix timeline:** Depends on severity (critical = ASAP)

## Scope

**In scope:**
- Bypass techniques that evade detection
- False negatives (attacks that aren't caught)
- API vulnerabilities
- Data leakage issues

**Out of scope:**
- False positives (legitimate prompts flagged) - report as regular issues
- Rate limiting complaints
- Feature requests
